[Component Driven User Interfaces](https://www.componentdriven.org/) -- практика разработки и проектирования пользовательских интерфейсов с помощью модульных компонентов. Пользовательские интерфейсы строятся «снизу вверх», начиная с базовых компонентов, которые комбинируются в составные компоненты и, наконец, в страницы. Ссылки внутри.
